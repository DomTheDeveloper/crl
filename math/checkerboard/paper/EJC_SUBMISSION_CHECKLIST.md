# Electronic Journal of Combinatorics submission checklist

Status: internal preparation only. Do not submit or contact editors until the author explicitly approves.

## Initial submission

- [x] Original, self-contained combinatorics manuscript.
- [x] PDF compiles without LaTeX errors.
- [x] Abstract states the background and principal theorem directly.
- [x] Proof is human-readable; Lean is supplementary verification rather than a substitute for exposition.
- [x] AI-assistance disclosure is included.
- [x] Prior work is cited and a separate prior-art audit is present.
- [x] Independent finite checker is included.
- [x] Exact-arithmetic certificate checker is included.
- [ ] Author manually reads every sentence and checks every displayed identity.
- [ ] Author chooses the final affiliation: Independent researcher or an institutional affiliation authorized for this work.
- [ ] Author supplies the submission email and ORCID, if desired.
- [ ] Author confirms the manuscript is not under consideration elsewhere.
- [ ] Author approves release of the repository and archival snapshot.
