# Math migration complete

All math-related code and branches formerly developed in the temporary app repository have been copied into this repository. CRL is the sole canonical home for that work.

The old repository is not required for builds, verification, source links, or continued development and may be deleted.

Canonical branch snapshots are stored under `migrated/test-expo-app/<original-branch>`, with original branch-name aliases retained in CRL where available. Historical source commit identifiers are recorded only as provenance; no active workflow fetches the old repository.
